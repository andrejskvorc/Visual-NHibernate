ArchAngel Release History
Known Issues in latest build
ArchAngel Editor
  • Previewing a function's text output is slow
  • Deleting a function doesn't close the function's tab
  • Import Wizard for CodeSmith and MyGeneration templates not complete yet
ArchAngel Workbench
  • Purpose of the 'Setup Model' button not adequately described
  • 'Load Graph' button not working
  • Only SQL Server 2000, SQL Server 2005 and SQL Server Express currently supported
  • Merging is not complete: Overwrite files by clicking the 'Overwrite All' button instead of clicking the 'Finish' button
  • SQL 2005's new datatypes VARBINARY and XML not catered for yet
  • If some tables are deselected then some files cannot be generated due to errors
  • Program hangs at generation time if a database table column links to itself
General
  • Retrieval of license not automated - internet activation
  • Help files not complete
  • Task pane help for individual screens not complete
